<h1 align="center">Yuki Baileys</h1>

Yuki Baileys is a library for interacting with WhatsApp Web via WebSockets.  
You don't need a web engine like Selenium or Chromium.

Please note that this is a **fork** created to fulfill the needs of the Yuki WhatsApp Script Bot.  
Updates will follow the original Baileys upstream:  
https://github.com/WhiskeySockets/Baileys

</div>

> [!WARNING]
> Usage instructions are not provided in this repository.  
> Please refer to the official [Baileys documentation](https://github.com/WhiskeySockets/Baileys)  
> or the [Official Yuki WhatsApp Bot](https://github.com/OhMyDitzzy/Yuki) for more information.
